# nvim-config
nvim config for develop vuejs + typescrip (but you can add any lsp)

## keybindings

**, + o + t** = toggle the file tree at left

**Ctrl + e** = jumps between the editor and de file tree

**, + f + f** = find files by filename
 
**, + f + g** = find text in all the files in folder

**, + f + h** = search text in opened file

**, + f + r** = starts search and replace in opened file
" quita el molesto higligh despues de una busqueda
**, + n + h** = turn off the highlight after a search

## autocomplete

use **Enter** to acept the COC suggestion
user **Tab** or **Shift + Tab** to navigate trow COC suggestions 


## Dependencies

sudo apt-get install **ripgrep**: for the greap text search (, + t + g)
**vim-plug:** for install required plugins
**latest Nvim:** of course
## Start
1.- put init.vim file in route **/home/<YourUser>/.config/nvim/init.vim**
2.- Open nvim and run **:PlugInstall**
3.- enjoy
